# React 笔记

## 脚手架


```bash
# 安装
npm install -g create-react-app

# 创建React脚手架工程
create-react-app react_test

# 开启服务器
yarn start

# 前端生成静态文件
yarn build

# 
yarn test

# 
yarn eject

```
